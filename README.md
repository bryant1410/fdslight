# IP层代理软件(VPN)，更多详情请查看项目上方的wiki页面
教程链接 https://fdslight.github.io/fdslight/

# 软件功能说明  
1. 支持UDP穿透技术，即P2P技术,(gateway模式)
2. TCP使用分发规则：在名单内的使用代理。
3. 可以指定特定局域网客户机进行UDP全局代理，该功能为游戏联机准备(gateway模式)
